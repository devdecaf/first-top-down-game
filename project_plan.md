Core Gameplay Loops

1. Moving around the map
2. Collecting gems from chests
3. Dodging or fighting enemies

Implementation Steps

1. Create a movable character with dust trail
2. Create enemy that follows character
3. Add health to character that lowers from enemies
4. Add hands to character that follows the mouse
5. Create a projectile and fire one when mouse is clicked
6. Add health and damage to enemies from projectile
7. Create interactable items
8. Add ability for character to pickup an item
9. Power up or count items that character has picked up
10. Add interactable chests that include items
